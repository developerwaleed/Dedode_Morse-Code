## Getting Started

## Run Locally

Clone the project

```bash
  git clone https://github.com/developerwaleed/Dedode_Morse-Code.git
```

Go to the project directory

```bash
  cd Dedode_Morse-Code
```

Install dependencies

[ruby](https://www.ruby-lang.org/en/documentation/installation/)

Run Script

```bash
  ruby morse_code.rb
```
